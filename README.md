# SureScore AI 🏥 🚗
**An AI-powered multi-agent system for automated insurance risk assessment.**

SureScore replaces the manual 2-to-5 day underwriting process with instant, machine learning-driven decisions. This repository contains the Machine Learning backend and the testing dashboard for the Health Insurance Agent.

## Tech Stack
* **AI Engine:** XGBoost, Scikit-Learn, Pandas
* **API Backend:** FastAPI, Uvicorn
* **Frontend:** HTML5, JavaScript, Tailwind CSS
